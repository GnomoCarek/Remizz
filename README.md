# Rmizz

O Remizz é um player de música robusto e intuitivo, desenvolvido para oferecer a melhor experiência na execução de arquivos MP3 locais. Com um design moderno e animações fluidas, ele vai além de um simples player, permitindo personalização total e organização inteligente da sua biblioteca.

Funcionalidades

Experiência Musical

Player Completo: Controles de reprodução avançados, incluindo modo aleatório (shuffle) e repetição.

Animações Dinâmicas: Interface viva com animações sincronizadas durante a execução das faixas.

Sistema de Busca: Encontre suas músicas rapidamente com um motor de busca integrado.

Organização Inteligente:

Filtro Anti-Spam: Sistema inteligente que identifica apenas músicas, ignorando automaticamente áudios de aplicativos de mensagens como o WhatsApp.

Seleção de Pastas: Você define exatamente em quais diretórios o aplicativo deve buscar seus arquivos de áudio.

Playlists e Favoritos: Crie coleções personalizadas e salve suas faixas preferidas para acesso rápido.

Personalização e Utilidades:

Modo Sleep (Timer): Defina um tempo para o aplicativo encerrar a reprodução automaticamente, ideal para quem gosta de dormir ouvindo música.

Temas Customizáveis: Altere as cores do aplicativo para deixar a interface com a sua cara.

Tecnologias Utilizadas:

Flutter / Dart: Framework principal para o desenvolvimento cross-platform.

Audio Service / Just Audio: (ou a biblioteca que você usou) para o gerenciamento robusto do player.

Provider / Riverpod: (ou sua gerência de estado) para controle de temas e playlists.
